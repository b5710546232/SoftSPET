# SOFTSPET

SOFTSPET is a project, part of Software Specification & Design Lab.

### Usecase
* Give food to pet.
* Cure the Pet.
* Switch on/off the light.
* Play mini game with Pet.
* Expression the emotion of pet.
* Clean the Pet.
